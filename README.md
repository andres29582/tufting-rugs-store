# Tufting Rugs Store

Sistema web para apresentação, personalização e solicitação de compra de alfombras tufting feitas sob encomenda.

## Objetivo do projeto

O objetivo deste projeto é desenvolver uma plataforma para um negócio de alfombras personalizadas, permitindo que clientes visualizem modelos disponíveis, solicitem orçamentos e enviem ideias personalizadas para criação de novas peças.

## Funcionalidades planejadas

- Portfólio de alfombras disponíveis
- Página de detalhes de cada produto
- Solicitação de orçamento personalizado
- Entrada de texto para descrição da ideia do cliente
- Geração de imagem com apoio de IA
- Cálculo inicial de preço
- Cadastro de pedidos
- Área administrativa para gerenciar modelos, pedidos e preços

## Tecnologias previstas

- Frontend: Next.js, TypeScript e Tailwind CSS
- Backend: Node.js ou NestJS
- Banco de dados: PostgreSQL
- IA: integração futura para geração de imagens personalizadas
